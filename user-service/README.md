# User Service

## Overview

This is the User Service, a part of the Event Planner application. This service is responsible for managing user data.

## Features

- Create a new user
- List all users
- Delete a user

## Technologies Used

- Node.js
- Express
- SQLite

## How to Run Locally

\`\`\`bash
cd user-service
npm install
npm start
\`\`\`
